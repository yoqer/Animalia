# ** Creamos Compatibilidad para Agentes de Escritorio Asistentes Personales **

Compatibilidad para entrenamiento y uso con Asistentes Open Source del estilo del antiguo tipico AI Assistent, como OpenCraw, el Pingüino y semejantes que puedan usar nuestras IAs locales reentrenadas, por conexion con API y similares.


Para entrenamiento, Fine-tuning de modelos con bases de datos, archivos locales como almacenajes de usuarios.
Todo posible en Local, con una intuitiva Interface Grafica y completa privacidad de sus datos.
 
Tambien puede usar las versiones avanzadas para entrenar la Comunicacion Multiespecie desde la Aplicacion 
de AlIAmAlIA.  [WAI.Cam](http://wai.cam)


Tambien recomendamos la Contenerizacion de estos Asistentes para su aislado, o personalizaciones con exposicion de API, para entrenamientos de actividad. Del estilo [Pingüino](http://GitHub.com/yoqer/pingüino) o el [Garras](http://GitHub.com/Trompetilla/Garras) de forma OpenCraw.

Tambien puedes incluir la personalizacion de tu implementacion de Asistente Local, desde tu usuario; [Mira aqui la nuestra, para usar OpenCraw y Garras con su despliegue aislado con  memoria persistente en contenedores seguros Docker:](https://github.com/yoqer/Animalia/blob/Entreno/INVESTIGACION_MEMORIA_ASISTENTES_IA_1.md)
